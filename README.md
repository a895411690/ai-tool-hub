```

├── src/                      # 源代码目录
│   ├── css/                 # 样式文件
│   │   └── main.css        # 主样式 (30行)
│   ├── js/                  # JavaScript 文件
│   │   └── app.js          # 主逻辑 (1,184行)

├── package.json             # 项目配置
├── vite.config.js           # Vite 配置
└── tsconfig.json            # TypeScript 配置
```

## 📖 使用指南

1. **访问网站**：直接点击在线预览链接
2. **搜索工具**：顶部搜索框支持名称、拼音首字母搜索
3. **分类筛选**：点击分类标签查看特定领域工具
4. **收藏工具**：点击工具卡片右上角星星图标
5. **查看收藏**：点击导航栏「我的收藏」
6. **复制链接**：点击工具卡片链接图标一键复制
7. **随机探索**：点击「随机推荐」发现新工具
