🔍 问题诊断

原始问题： 首页显示白色背景而不是赛博朋克黑色背景

根本原因： 根目录 index.html 使用了错误的配置：

  • 没有链接 main.css 文件
  • 使用 Tailwind 的 bg-dark-900（可能未定义）
  • 缺少赛博朋克背景样式

修复方案： 同步了根目录 index.html 与 src/index.html 的配置

📁 文件修改详情

修改的文件： index.html（根目录）

修改内容：

// 添加 CSS 链接
<link rel="stylesheet" href="./css/main.css">

// 修复 body 类
<body class="text-gray-100 min-h-screen tech-bg">

新创建的 CSS 文件： ./css/main.css

✅ 验证步骤

  1. 打开浏览器开发者工具（F12）
  2. 检查 Network 标签页，确认 main.css 成功加载
  3. 检查 Elements 标签页，查看 body 元素的样式
  4. 确认背景显示为赛博朋克风格（黑色渐变 + 扫描线）
  5. 如果没有显示正确，清除浏览器缓存后重试
