🎯 设计原则

  • 避免纯白色：减少视觉疲劳，增强赛博朋克氛围
  • 层次分明：不同重要性的内容使用不同颜色
  • 一致性：整个网站使用统一的颜色方案
  • 可读性：确保所有颜色在深色背景上清晰可读
  • 品牌感：青色系符合AI和科技的主题

📋 CSS更新摘要

/* 主文本颜色 - 浅青色 */
body { color: #e0f7fa !important; }

/* 标题颜色 - 青色 */
h1, h2, h3, h4, h5, h6 { color: #00ffff !important; }

/* 次要文本 - 浅灰色 */
p, span, li { color: #b0bec5 !important; }

/* 链接颜色 - 中青色 */
a { color: #80deea !important; }

/* 禁用纯白色 */
.text-white { color: #e0f7fa !important; }
