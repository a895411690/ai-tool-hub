# AI 简历优化工具

> AI驱动的简历优化工具，支持实时预览、PDF导出和AI优化建议。

## 🚀 功能特性

- ✨ **实时预览** - 编辑时即时预览简历效果
- 🤖 **AI 优化** - 智能简历优化建议（模拟/真实 API）
- 📄 **PDF 导出** - 一键生成专业 PDF 简历
- 💾 **本地存储** - 自动保存到浏览器
- 📱 **响应式设计** - 支持移动端和桌面端
- 🎨 **深色主题** - 舒适的编辑体验

## 📖 使用方法

1. 打开 `index.html` 文件
2. 在左侧填写简历信息
3. 右侧实时预览效果
4. 点击"AI 优化"获取建议
5. 点击"导出 PDF"下载简历

## 🛠️ 技术栈

- HTML5 + Tailwind CSS
- Vanilla JavaScript
- html2pdf.js (PDF生成)
- 模拟 AI API（可替换为真实 API）
